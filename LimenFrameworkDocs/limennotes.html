FLimenNotesModule

Manages the lifecycle of the Limen Notes module within the Unreal Engine.

Overview

The FLimenNotesModule class is a module interface for the Limen Notes plugin. It overrides the StartupModule and ShutdownModule functions to handle initialization and cleanup tasks when the module is loaded and unloaded, respectively.
