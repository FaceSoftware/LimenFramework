UDialoguePlayerBase

Base class for managing dialogue playback, providing a framework for playing, finishing, and stopping dialogues.

Overview

UDialoguePlayerBase is an abstract base class designed to manage the lifecycle of dialogue playback. It includes methods for playing, finishing, and stopping dialogues, as well as subscribing to events when a dialogue finishes. This class ensures that all necessary cleanup is handled properly by overriding BeginDestroy.

Usage & Implementation Notes

  • Threading Constraints: This class is not thread-safe and should be used on the game thread.

  • Replication Requirements: None.

  • Memory Lifetime Management: The DialogueData is managed using a weak pointer to avoid memory leaks. Ensure that the dialogue data remains valid for the lifetime of the dialogue player.

  • Initialization Dependencies: The PlayerId is initialized in the constructor, ensuring each instance has a unique identifier.
