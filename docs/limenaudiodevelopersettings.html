ULimenAudioDeveloperSettings

Manages developer-specific settings for the Limen audio module.

Overview

ULimenAudioDeveloperSettings is a subclass of ULimenModularSettingsSubsystemDeveloperSettings. It configures developer-specific settings related to the Limen audio module. This class sets up a configuration section named " Limen - Audio Settings".

Properties

Type

Name

Access

Description

FString

SectionName

Public

The name of the configuration section, set to " Limen - Audio Settings".

Functions

No significant functions are defined in this class.

Usage & Implementation Notes

  • This class is used to manage developer-specific settings for the Limen audio module.

  • It inherits from ULimenModularSettingsSubsystemDeveloperSettings and sets up a configuration section.
