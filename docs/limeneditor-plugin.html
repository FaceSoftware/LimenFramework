LimenEditor Plugin

WARNING: This Module or Plugin is currently experimental and under active development. It has not been fully tested for production environments and may contain bugs or breaking API changes.

Introduction

The LimenEditor plugin is a comprehensive toolkit designed to enhance the development and management of Unreal Engine projects within the Limen Framework ecosystem. It provides essential modules for managing the lifecycle of the editor, simplifying collision proxy generation, and facilitating easy integration of custom plugins or extensions.

Included Modules

Module Name

Description

LimenEditor Module

Manages the initialization and cleanup of the Limen Editor plugin.

LimenLevelCollisionProxy Module

Simplifies the management and generation of collision proxies within Unreal Engine projects.

These modules work together to streamline development processes, ensuring that all necessary resources are properly managed and configured for optimal performance and ease of use.
