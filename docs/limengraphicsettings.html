FLimenGraphicSettingsModule

Manages the initialization and shutdown of the Limen Graphic Settings module.

Overview

The FLimenGraphicSettingsModule class is a module interface for managing the lifecycle of the Limen Graphic Settings module in Unreal Engine. It overrides the StartupModule and ShutdownModule functions to handle the initialization and cleanup processes, respectively.

Usage & Implementation Notes

  • This module does not perform any specific initialization or cleanup tasks at this time, as indicated by the empty implementations of StartupModule and ShutdownModule.
