FLimenTriggersModule

Manages the initialization and shutdown of the LimenTriggers module within the Unreal Engine.

Overview

The FLimenTriggersModule class is a module interface for the LimenTriggers plugin. It overrides the StartupModule and ShutdownModule functions to handle the initialization and cleanup of the module, respectively.
