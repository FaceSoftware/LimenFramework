FLimenCreditsModule

Manages the lifecycle of the Credits module within the LimenCore plugin.

Developer Settings

Class Name

Role

ULimenCreditsDeveloperSettings

Manages developer settings for the Limen Credits module, allowing customization of credit sections.

Design Philosophy

The LimenCredits Module is designed to provide a robust and flexible framework for managing credits within a plugin system. It ensures that credits are displayed correctly and efficiently while providing developers with tools to customize their experience.

Recommended Integration

  • Developers can use the FLimenCreditsModule to integrate credits into their plugins seamlessly.

  • The ULimenCreditsDeveloperSettings class allows for easy customization of credit sections, making it a versatile tool for various plugin needs.
